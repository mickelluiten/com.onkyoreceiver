Onkyo Commands Mappings syntax :  zone.command=argument			
Example :  main.power=on			
Example :  zone2.volume=22
	
No guarantee all commands works depends on the receivertype	
	
	
	zone 		: main, zone2 or zone3
	command 	: It is the name directely under the raw EISCP command ( PWR --> power)
	arguments	: It is the name under the vaules ( '00' --> standy)
	
Also possible to send RAW EISCP command like PWR00 for power OFF or PWR01 for power ON
